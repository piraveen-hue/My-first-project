# My-first-project-programming in C
Student Grade Calculator
📌 Description

This is a simple C program that calculates a student's total marks, average, and grade based on marks entered for 5 subjects.

🎯 Features
Takes student name as input
Accepts marks for 5 subjects
Calculates total marks
Calculates average
Displays grade (A, B, C, D, F)
🛠️ Technologies Used
C Programming
Standard Input/Output functions
▶️ How It Works
User enters student name
User enters marks for 5 subjects
Program calculates total and average
Program assigns grade based on average
Displays the result
📊 Grade Criteria
Average Marks	Grade
90 and above	A
75 – 89	B
60 – 74	C
50 – 59	D
Below 50	F
